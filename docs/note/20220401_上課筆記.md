# 切版基礎

## 快速製造標籤 (快捷鍵)

div (enter)

div#container (enter)

div#container.w-screen.h-screen (enter)

## 認識 Margin、Padding、Border

### Margin

對外的元素關係,跟別人之間的距離

mx (左右)  my (上下)  m (全部)  mt (上)  mr (右)  mb (下)  ml (左)

### Padding

外框到元件本體之間的距離

### Border

外框

## 快捷鍵

一次兩個 p. 可以打 p.*2 (enter)

## 全部置中

flex items-center justify-center

## Flex 屬性

將 block 變成橫的排或直的排

## grid 

可以進行等分,將每一個格子的大小統一
